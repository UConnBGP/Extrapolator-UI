Policy Descriptions

Enforce Invalid ASN Only

Block route announcements only when the originating AS is in conflict with a Route Origin Authorization (ROA). Announcements containing prefixes exceeding the maximum length authorized by a ROA are allowed.

Enforce Invalid Length Only

Block route announcements only when the length of the announced prefix exceeds the maximum length authorized by a ROA. Announcements originating from an AS that is not authorized to do so are allowed.

ROV

Block route announcements using standard Resource Certification (RPKI) Route Origin Validation (ROV) as defined in RFC 6811.
